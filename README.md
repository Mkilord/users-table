# Users Table App

## Описание

Приложение на React с использованием Material UI.
Отображает таблицу пользователей с возможностью:

* сортировки по колонкам
* фильтрации
* пагинации
* открытия модального окна с подробной информацией о пользователе

Проект демонстрирует работу с MUI-компонентами, управлением состоянием и компоновкой интерфейса.

---

## Стек технологий

* React
* Material UI (MUI)
* JavaScript (ES6+)
